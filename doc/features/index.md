# CRM Web Application Features

This section provides detailed documentation on all the features available in the CRM Web Application.

## Core Features

The CRM Web Application includes the following core features:

1. [**User Authentication and Authorization**](./authentication.md)
   - Secure login with credentials or Google accounts
   - Role-based access control
   - Session management

2. [**Google Services Integration**](./google-integration.md)
   - Google Drive integration for document management
   - Gmail integration for email communications
   - Google Calendar integration for appointment scheduling

3. [**Ticket Management**](./tickets.md)
   - Create, read, update, and delete support tickets
   - Assign tickets to employees
   - Track ticket status and resolution
   - Email notifications for ticket updates

4. [**Lead Management**](./leads.md)
   - Create, read, update, and delete sales leads
   - Track lead status and conversion
   - Schedule follow-up activities
   - Attach relevant documents

5. [**Contract Management**](./contracts.md)
   - Create, read, update, and delete contracts
   - Track contract status and terms
   - Manage contract attachments
   - Set up contract expiration notifications

6. [**Email Templates & Campaigns**](./email-templates.md)
   - Create and manage email templates
   - Design email campaigns
   - Track email delivery and engagement
   - Automate email notifications

7. [**Dashboard & Reporting**](./dashboard.md)
   - Overview of key metrics
   - Customizable dashboards
   - Data visualization
   - Export reports in various formats

8. [**User Management**](./user-management.md)
   - Create and manage user accounts
   - Assign user roles and permissions
   - Manage user profiles and preferences
   - Reset passwords and manage account security
