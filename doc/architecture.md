# CRM Web Application Architecture

This document describes the architecture of the CRM Web Application, including its major components, technology stack, and database schema.

## System Overview

The CRM Web Application is built using a standard three-tier architecture:

1. **Presentation Layer**: Thymeleaf templates, HTML, CSS, JavaScript
2. **Application Layer**: Spring Boot MVC, Service Components
3. **Data Layer**: Hibernate ORM, MySQL Database
