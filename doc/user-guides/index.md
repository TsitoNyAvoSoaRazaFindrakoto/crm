# CRM Web Application User Guides

This section provides comprehensive user guides for different roles within the CRM Web Application. Each guide is tailored to the specific needs and permissions of each role.

## Available User Guides

### [Manager Guide](./manager.md)

The Manager Guide covers all features available to users with the Manager role. Managers have the highest level of access and can perform administrative tasks such as:

- Creating and managing user accounts
- Assigning roles and permissions
- Viewing and managing all tickets, leads, and contracts
- Accessing comprehensive reports and analytics
- Configuring system settings
- Managing email templates and campaigns

### [Employee Guide](./employee.md)

The Employee Guide covers features available to standard employee users. Employees can:

- Manage their assigned tickets, leads, and contracts
- Create new tickets and leads
- Communicate with customers
- Schedule appointments
- Upload and share documents
- Track their performance metrics

### [Customer Guide](./customer.md)

The Customer Guide explains how customers can use the customer portal. Customers can:

- View their tickets, leads, and contracts
- Create new support tickets
- Communicate with their assigned representatives
- Access shared documents
- Update their contact information and preferences

## Getting Help
