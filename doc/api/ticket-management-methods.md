# Ticket Management Methods

### createTicket()
- Purpose: Open a new support ticket
- Parameters: title, description, priority, …

### updateTicket()
- Purpose: Modify existing ticket details
- Parameters: ticketId, status, assignedTo, …

### closeTicket()
- Purpose: Finalize a resolved issue
- Parameters: ticketId

### reopenTicket()
- Purpose: Reactivate a resolved ticket if the issue recurs
- Parameters: ticketId
