# User Management Methods

### createUser()
- Purpose: Create a new CRM user account
- Parameters: username, email, role, …  

### editUser()
- Purpose: Modify existing user details
- Parameters: userId, updated fields  

### deactivateUser()
- Purpose: Remove login access while preserving user data
- Parameters: userId  

### activateUser()
- Purpose: Restore login access for inactive accounts
- Parameters: userId  

### deleteUser()
- Purpose: Permanently remove user data
- Parameters: userId
